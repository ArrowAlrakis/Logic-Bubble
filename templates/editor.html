How to use

1. Submit textType or paste text in the Input area, click Submit.
2. Extract bubblesSelect any phrase in Source Text, press Enter or click the tooltip.
3. ConnectPress S to enter Connect mode. Click a bubble to start, click another to draw an arrow.
4. Merge bubblesDrag one bubble onto another — when they overlap, release to merge their text.
5. Save / ExportCtrl+S to save. File → Export PNG or Export JSON to download.
Import formatJSON must contain title, source_text, and graph_data fields.
